#  Supported Algorithms and Classes

wolfJSSE currently provides implementations of the following JSSE classes:

- SSLContext
    + TLS 1.0, TLS 1.1, TLS 1.2, TLS 1.3
- SSLEngine
- SSLSession
- SSLSocket
- SSLServerSocket
- SSLSocketFactory
- SSLServerSocketFactory
- KeyManagerFactory
- X509KeyManager
- TrustManagerFactory
- X509TrustManager
- X509Certificate
